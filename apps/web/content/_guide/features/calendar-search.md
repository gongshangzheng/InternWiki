## 日历

[日历页](/calendar)汇总你的所有有日期任务和周期任务。切换实习生下拉框查看不同人的日程。

### 功能说明

- **任务事件**：有 `startDate` 的任务会显示在日历上，颜色对应任务状态
- **周期任务**：设置了 `recurring` 的任务会按周期重复显示
- **实习生切换**：顶部下拉框切换查看不同实习生的日历
- **URL 分享**：参数 `?intern=xxx` 可以直接分享特定实习生的日历链接

### 日历视图

支持月视图、周视图和列表视图，可通过顶部按钮切换。

## 搜索

按 **⌘K**（Mac）或 **Ctrl+K**（Windows）随时打开搜索，覆盖日报、周报、月报、文档、项目和实习生。

### 搜索范围

| 类型 | 说明 |
|------|------|
| 日报 | 搜索所有实习生的日报标题和内容 |
| 周报 | 搜索周报标题和内容 |
| 月报 | 搜索月报标题和内容 |
| 文档 | 搜索技术文档 |
| 项目 | 搜索项目名称和描述 |
| 实习生 | 搜索实习生姓名和团队 |

搜索结果按相关性排序，点击即可跳转到对应详情页。
