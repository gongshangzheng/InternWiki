## 欢迎加入

欢迎来到 InternWiki 团队！这篇指南帮助你快速上手。

## 第一周清单

1. 配置开发环境（Node.js 25 + pnpm）
2. 克隆仓库并运行 `pnpm install && pnpm dev`
3. 阅读项目 PLAN.md 和 README.md
4. 创建你的实习生空间（`pnpm report new-intern`）
5. 写第一篇日报

## 开发规范

- 提交信息使用约定式格式（Conventional Commits）
- PR 需要至少一人 Review
- 日报每天下班前提交
- 周报每周五提交
- 月报每月最后一天提交
