## 项目背景

InternWiki 前端需要一套统一的组件库，保证各页面视觉一致性，并提高开发效率。

## 技术方案

- **框架**: React 19 + TypeScript
- **样式**: Tailwind CSS v4 + class-variance-authority
- **图标**: lucide-react
- **测试**: Playwright

## 当前阶段

基础组件已完成，正在开发业务组件（搜索弹窗、报告列表等）。
