## 1. 实习生仪表盘（`/interns/:name`）

实习生个人主页，展示综合数据：

- **日报热力图**：26 周回溯，GitHub 风格贡献网格，颜色深浅表示当天日报数量
- **任务进度条**：异步加载各项目任务树，显示 completed/total 百分比进度条
- **统计卡片**：日报数、周报数、月报数、项目数
- **最近报告列表**：混合展示日报/周报/月报，按日期倒序
- **项目卡片**：展示项目状态、标签、进度
- **快捷入口**：日报/周报/月报/文档/项目 5 个快捷链接

## 2. 习惯追踪（`/interns/:name/habits`）

从日报的 `- [x] 习惯名 #tag` 自动解析习惯数据：

- **连续打卡天数**：当前连续多少天完成该习惯
- **最长连续天数**：历史最高连续记录
- **完成率**：30 天内的完成百分比
- **30 天热力图**：绿色=完成，红色=未完成，灰色=无记录
- **今日打卡完成率**：今天所有习惯的完成比例

数据来源是日报 Markdown 中的 `## 习惯打卡` 章节，格式详见 `content-creation.md` §8。

## 3. 日历（`/calendar`）

基于 FullCalendar 6 的任务日历：

- **实习生选择器**：URL 参数 `?intern=xxx` 同步，可直接分享链接
- **五种视图**：月视图 / 周视图 / 3日视图 / 日视图 / 列表视图
- **中文本地化**：周一为首日，中文按钮和日期格式
- **侧边栏**：
  - **待办 Tab**：逾期任务（红色标记）、今日待办、无日期任务
  - **已完成 Tab**：近 30 天完成的任务
- **任务详情弹窗**：点击任务显示详情，含描述、标签、笔记链接、关联日报链接
- **周期任务展开**：`recurring` 配置的任务自动在日历上生成多个实例

### 日历分类颜色

任务的 `category` 字段决定日历上的事件颜色：

| category | 颜色 | 适用 |
|----------|------|------|
| `study` | 紫色 | 学习、培训 |
| `health` | 绿色 | 运动、健康 |
| `work` | 蓝色 | 工作、开发 |
| `social` | 橙色 | 社交、会议 |
| `life` | 粉色 | 生活、日常 |
| `other` | 灰色 | 其他 |

## 4. 搜索（`⌘K` / `Ctrl+K`）

全局搜索弹窗：

- **触发方式**：导航栏点击「搜索」按钮，或按 `⌘K`(Mac) / `Ctrl+K`(Windows)
- **搜索范围**：日报、周报、月报、技术文档、项目、实习生
- **模糊匹配**：支持不精确匹配，按相关度排序
  - 完全匹配：100 分
  - 前缀匹配：90 分
  - 包含匹配：70 分
  - 模糊匹配（字符顺序匹配）：40 分
- **键盘导航**：↑↓ 选择结果，回车跳转，ESC 关闭
- **结果限制**：最多显示 20 条，按分数排序

## 5. 使用指南（`/guide`）

导航栏「指南」入口（HelpCircle 图标），面向实习生的站内使用说明页面。

包含：项目目标、Agent 协作理念、分支工作流、报告写作规范、项目管理、CLI 速查、习惯打卡、搜索说明等内容。
